Control de Resguardos Contraloría

2017PHPMySQLHTML

Desarrrollo

La Secretaría de Contraloría y Transparencia Gubernamental, se da el área llamada recursos materiales y servicios generales. Esta se encarga en atender las necesidades que surgen, para realizar adecuadamente los diferentes trabajos de las áreas que conforman esta Secretaría, es así, que este departamento se convierte en el intermediario para gestionar ante la Secretaría de Finanzas del Gobierno del Estado el equipo y materiales necesarios para el buen desarrollo de los trabajos de las diferentes áreas, como son computadoras mesas, sillas secretariales, impresoras, escritorios, equipo de impresión, etc. Asignándose a cada personal de acuerdo a las necesidades planteadas, como también a dar de baja y cambios de equipo de trabajo deficientes, todo esto se llevaba a cabo de manera convencional, realizándose su control a través de hojas de cálculo para verificar que equipo de trabajo estaba asignado y dado de baja. Por lo anterior al área de recursos materiales se le hacía más difícil el manejo de esto, por lo que se solicitó el desarrollo de un sistema local inventarial que permita un control adecuado y confiable para el manejo de toda esta información.

Problemas a considerar:
  • Se pretende ayudar para el buen funcionamiento de la secretaría de contraloría y transparencia gubernamental, específicamente el área de recursos materiales y servicios generales.
  • Tener un inventario digital de bienes muebles y equipos de cómputo actualizado, que permita consultar las características físicas (marca, modelo, serie, color, etc.)
  • Igualmente número de factura de compra, área a donde se asignó y el nombre de la persona quien resguardo dicho bien, así como las condiciones actuales en las que se encuentra dicho material.

Marco de Trabajo

Debido a que los requerimientos era muy parecidos al proyecto del CRIE, se tomo la decisión de tomar como base el proyecto anterior y desarrollar este nuevo proyecto, lo que redujo bastante el tiempo de desarrollo al reutilizar el código anterior.

Tecnologías Usadas:
  • PHP:

    Se utilizó PHP como lenguaje de servidor, para atender las consultas de parte del cliente.

  • MySQL:

    Como gestor de Base de datos.

  • HTML:

    Como principal tecnología para la maquetación de las paginas.

  • Boostrap:

    Para obtener un diseño responsive con las clases que este framework ofrece.

  • Javascript:

    Para dar mayor funcionalidad al sistema

  • Otras:

    Se utilizaron otras tecnologías como son JQuery, Ajax, y librerías como: PhpMailer, fpdf etc.
