Control de Expedientes CRIE con Laravel

2020LaravelPHP

Desarrrollo

El proyecto es exactamente igual con los requisitos expuestos en la primera versión del proyecto, pero en este caso utilizando un framework de desarrollo Laravel, mejorando considerablemente muchas de las funciones que tenia anteriormente la aplicación, sobre todo en lo que respecta al tema de seguridad, debido a que Laravel ya cuenta con funciones establecidas y probadas en lo que respecta a este tema, también se mejora bastante el tema de la comunicacion, en la versión anterior no se tenia un patrón de diseño establecido, Laravel por si, utiliza el patrón de diseño MVC(Modelo Vista Controlador).

Marco de Trabajo

Se muestran a continuación algunas de las tecnologías usadas en el desarrollo del proyecto.

Tecnologías Usadas:
  • Laravel:

    Framework de desarrollo para PHP que utiliza el patrón de diseño MVC.

  • PHP:

    Se utilizó PHP como lenguaje de servidor, para atender las consultas de parte del cliente.

  • MySQL:

    Como gestor de Base de datos.

  • HTML:

    Como principal tecnología para la maquetación de las páginas.

  • Otras:

  • MySQL
  • Netbeans
