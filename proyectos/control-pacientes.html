Control de Pacientes

2020JavaJSPsServletsMySQL

Desarrollo

Para poner en práctica lo aprendido sobre la parte empresarial de Java, en el cual incluye Servlets y JSPs, se aprovechó la necesidad de llevar el control de pacientes de un consultorio dental, los requisitos eran mínimos, por lo cual quedó perfecto para desarrollar lo aprendido, como se observa el proyecto consiste solamente en agregar pacientes a una base de datos, modificar y eliminar en caso de ser necesario, funciones básicas de un CRUD. La aplicación aplica el patrón de diseño MVC(Modelo Vista Controlador), como vista tenemos al JSP, el cual hace una petición al servidor, que es procesada por el controlador que este caso es un Servlet, este procesa la petición y la manda al modelo que en esta caso es un JavaBeans, el cual accede a la base de datos para el cual se utilizó MySQL.
